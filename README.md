## Requerimientos

### Variables del Formulario

- **_formKey**: `GlobalKey<FormState>` - Clave para validar el formulario
- **_nombreController**: `TextEditingController` - Controlador para el campo de nombre completo
- **_emailController**: `TextEditingController` - Controlador para el campo de correo electrónico
- **_passwordController**: `TextEditingController` - Controlador para el campo de contraseña
- **_confirmPasswordController**: `TextEditingController` - Controlador para confirmar la contraseña
- **_ocultarPassword**: `bool` - Variable para mostrar/ocultar la contraseña
- **_ocultarConfirmPassword**: `bool` - Variable para mostrar/ocultar la confirmación de contraseña
